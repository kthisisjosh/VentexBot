</p>

## Getting Started
1. Clone the repo locally
2. Inside the repo run `npm install`
3. Create an `auth.json` file in `./src` as described below
4. Run `nodemon bot.js`

## auth.json example
```js
{
  "token": "<discord bot user token>",
}
```
